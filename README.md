# Todo Backend API

A simple restful Todo application backend built using Node.js, Express, and MongoDB.
It supports creating, reading, updating, and deleting todo tasks.

# api endpoints

GET /   -> GET ALL TODOS
GET / id -> GET A TODO BY ITS ID
POST / -> CREATE A NEW TODO 
DELETE / id -> DELETE A TODO BY ITS ID
PUT /id -> CHANGE THE COMPLETED STATUS OF A TODO ( TRUE TO FALSE , FALSE TO TRUE )
